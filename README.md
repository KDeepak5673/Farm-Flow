# Farm-Flow

A lightweight web app for sharing and saving farming techniques, built with a React frontend and an Express/MongoDB backend.

Demo: https://epics-farm-flow.vercel.app/

## Overview
- Purpose: Share, browse, and save practical farming techniques.
- Audience: Farmers, agritech students, and educators.

## Features
- Create, view, and save techniques.
- User authentication and profile management.
- Responsive UI and RESTful API.

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Getting Started
1. Install dependencies:

	```bash
	cd backend && npm install
	cd ../client && npm install
	```

2. Configure environment variables in `backend/.env` (e.g. `MONGO_URI`, `JWT_SECRET`).

3. Run the backend and client:

	```bash
	# in backend
	node server.js

	# in client
	npm start
	```
